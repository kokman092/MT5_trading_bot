## Installation Guide

1. System Requirements:
   - Windows 10 or later
   - Python 3.8+
   - MetaTrader 5 platform
   - Minimum 8GB RAM recommended

2. Setup Steps:
   ```bash
   # Install required packages
   pip install -r requirements.txt
   
   # Configure your settings in config.py
   # Start the bot
   python run_trader.py
   ```
